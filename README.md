Веб-приложение для фонового сбора HTML-контента со сторонних ресурсов по ключевым словам. Поддерживает стриминг прогресса загрузки и оффлайн-просмотр сохраненного контента.

## 🚀 Функционал
- **Поиск по ключевым словам:** База ключевых слов на сервере (`news`, `code`, `search`).
- **Стриминг с прогресс-баром:** Использование технологии **Server-Sent Events (SSE)** для отображения реального прогресса загрузки в байтах и процентах.
- **Оффлайн-архив (LocalStorage):** Хранение контента в браузере с возможностью просмотра через изолированный `iframe` без участия сети или сервера.
- **Обработка ошибок:** Корректный перехват и вывод ошибок (неверное слово, сбой сети, переполнение хранилища).

---

## 🛠 Стек технологий
- **Backend:** Node.js, Express, Axios
- **Frontend:** Pure HTML5, CSS3, Vanilla JavaScript (Fetch API, Server-Sent Events, LocalStorage)

## 💻 Локальный запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <ссылка_на_ваш_репозиторий>
   cd Http-colector
   ```
2. **Установите зависимости:**
   ```bash
   npm install
   ```
3. **Запустите сервер:**
   ```bash
   node .
   ```
4. **Откройте приложение в браузере:** [http://localhost:3000](http://localhost:3000)
